# Example Red Sand Mod

Copy this folder to your Godot user data mods directory:

```text
user://mods/example.red_sand/
```

Then open the in-game Mods screen, enable `Example Red Sand`, reload mods, and create a planet with the `Red Desert` preset.

This example reuses the core sand textures so it works without extra art. Replace the texture paths in `data/terrain_visuals.json` with mod-relative PNG paths when making a real terrain mod.
