# Run Keyframes 5 Direction Prompt

Using the provided astronaut turnaround as reference, create a pixel-art run keyframe sheet for game sprites.

Layout: 5 rows x 4 columns, one astronaut per cell, no text. Flat solid `#00ff00` chroma-key background only. Same scale and foot baseline in every cell.

Rows top to bottom:

1. front/down
2. front-right/down-right
3. right profile
4. back-right/up-right
5. back/up

Columns left to right:

1. run contact A
2. passing low
3. run contact B
4. passing high

Keep the same cream astronaut suit, black visor, orange rings, teal accents, square backpack, chunky pixel-art style, dark boots, small oval shadow.

Each row keeps the same facing direction across all 4 frames. In-place looping run animation, no drifting, no turning, no size changes, no labels, no gradient background, no blur.
