# FACTORIA 设计文档入口

本目录用于沉淀 FACTORIA 的产品设想、系统设计与玩法蓝图。

## 文档索引

- `project-vision.md`：项目愿景与体验定位，说明“异星工场 + vibe coding”的核心方向。
- `agent-blueprint-design.md`：agent 蓝图玩法设计，覆盖自然语言生成、修改蓝图与框选分析。
- `save-and-local-config-architecture.md`：存档、profile、本地配置与缓存的架构边界。

## 归档原则

1. 面向长期方向、系统边界、核心体验的内容放在 `docs/`。
2. 面向阶段执行、任务拆分、落地顺序的内容放在 `plans/`。
3. 面向 AI 协作约束、代码风格、任务流程的内容放在 `rules/` 或 `skills/`。
4. 文档先记录确定性方向；待玩法验证后，再补充数据结构、UI 流程与实现细节。
